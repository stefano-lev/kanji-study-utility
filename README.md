# Kanji Shishou

# Kanji Study Utility (漢字師匠)

A modern and interactive web application designed to help users study and master Japanese kanji effectively. This tool combines powerful features like flashcard quizzes, kanji dictionaries, and more to make kanji learning engaging and accessible.

## Features (Planned)
- **Flashcard Quiz:** Test your kanji knowledge by matching words or definitions with the correct kanji characters.
- **Kanji Dictionary:** Browse and filter kanji by stroke count, JLPT level, and more.
- **Stroke Order Quiz (Stretch Goal):** Practice drawing kanji with correct stroke order to reinforce muscle memory.

## Technologies
This project is being built using:
- **Frontend:** React
- **Backend:** Node.js with Express
- **Database:** MongoDB

## Project Goals
The goal of this utility is to provide a feature-rich, user-friendly platform for Japanese language learners at any level. Whether you're preparing for the JLPT or casually learning kanji, this app is designed with you in mind.
